第二章主要介绍排序，相当相当的重点

- 不管是实际生活还是程序，只要涉及到逻辑处理，排序绝对是必须的
- 几乎所有语言都会内置排序，一般优异的语言内置的排序都会按照使用情形来选择底层算法
- 算法基础面试必问

:::tip
在本章里将的所有排序，都是对数组进行排序
:::
